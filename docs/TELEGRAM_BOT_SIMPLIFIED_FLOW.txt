TELEGRAM BOT FLOW SIMPLIFIED - SUMMARY
=======================================

CHANGES MADE:
-------------

1. SIMPLIFIED BOOKING FLOW:
   - Reduced from asking 7+ questions to just 2 main questions:
     * Question 1: "Are you a new patient or existing patient?"
     * Question 2 (varies based on type):
       - New Patient: Full Name → Phone Number → Reason → Date/Time
       - Existing Patient: Phone/ID → Reason → Date/Time

2. REMOVED FIELDS FROM COLLECTION:
   - Age
   - Gender  
   - Email
   - Blood Group
   
   These are no longer asked during booking.

3. NEW PATIENT FLOW:
   /book → Patient Type Selection → Full Name → Phone → Reason → Date/Time → Confirmation

4. EXISTING PATIENT FLOW:
   /book → Patient Type Selection → Phone/ID → Reason → Date/Time → Confirmation

5. CONFIRMATION SCREEN NOW SHOWS:
   - Name
   - Phone
   - Reason
   - Date
   - Time
   - Doctor
   - Location

📊 WHAT THE TABLE SHOWS:
================================================================================

COLUMNS:
1. Medicine Name - Name + strength (if available)
2. Category - Medicine category
3. SKU - Stock keeping unit code
4. Stock - Number with colored badge
5. Status - Badge with icon (In Stock/Low Stock/Out of Stock)

EXAMPLE:
┌──────────────────┬────────────┬─────────┬────────┬──────────────┐
│ Medicine Name    │ Category   │ SKU     │ Stock  │ Status       │
├──────────────────┼────────────┼─────────┼────────┼──────────────┤
│ Paracetamol      │ Analgesic  │ PAR500  │  100   │ ✓ In Stock   │
│ 500mg            │            │         │        │              │
├──────────────────┼────────────┼─────────┼────────┼──────────────┤
│ Amoxicillin      │ Antibiotic │ AMO250  │   15   │ ⚠ Low Stock  │
│ 250mg            │            │         │        │              │
├──────────────────┼────────────┼─────────┼────────┼──────────────┤
│ Ibuprofen        │ Analgesic  │ IBU400  │    0   │ ⊘ Out of Stock│
│ 400mg            │            │         │        │              │
└──────────────────┴────────────┴─────────┴────────┴──────────────┘

================================================================================
🎨 STOCK BADGES:
================================================================================

GREEN BADGE (In Stock):
- Stock > Reorder Level (default 20)
- Example: Stock = 100, Reorder = 20 → Green

YELLOW BADGE (Low Stock):
- Stock > 0 AND Stock <= Reorder Level
- Example: Stock = 15, Reorder = 20 → Yellow

RED BADGE (Out of Stock):
- Stock = 0
- Example: Stock = 0 → Red

================================================================================
📝 BACKEND REQUIREMENT:
================================================================================

The backend /pharmacy/medicines endpoint MUST return:

```javascript
{
  "medicines": [
    {
      "_id": "abc123",
      "name": "Paracetamol",

      "strength": "500mg",
      "availableQty": 100,    ← THIS IS IMPORTANT!
      "reorderLevel": 20,
      ...
    }
  ]
}
```

OR

```javascript
[
  {
    "_id": "abc123",
    "name": "Paracetamol",
    "stock": 100,    ← OR THIS FIELD
    ...
  }
]
```

The frontend will check these fields in order:
1. availableQty
2. stock
3. quantity
4. Default to 0

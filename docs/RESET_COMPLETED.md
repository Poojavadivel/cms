## 🆘 Troubleshooting

### If You See Extra Users

The server automatically creates users from `.env` file on startup:
- `ADMIN_EMAIL=admin@hms.com`
- `DOCTOR_EMAIL=doctor@hms.com`

**To disable:** Comment out or remove these lines from `.env`

### If You Need to Reset Again

Simply run:
```bash
RESET_DATABASE.bat
```

Or manually:
```bash
cd Server
node reset_database_keep_users.js
```

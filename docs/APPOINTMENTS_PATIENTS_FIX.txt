   - min-height: 100vh (not height: 100vh)

WHY HEADERS WERE DISAPPEARING:
===============================

Problem 1: Body overflow:hidden
- Prevented all scrolling
- Content got clipped
- Headers pushed out of view

Problem 2: Container height:100vh
- Fixed height = no room for overflow
- Combined with overflow:hidden = content cut off
- Headers at top got hidden

Problem 3: No flex-shrink protection
- Headers could shrink to 0 height
- Disappeared when space constrained

THE FIX:
========

✅ Headers now have:
   - flex-shrink: 0 (can't shrink)
   - min-height: 60px (minimum size)
   - z-index: 5 (stays visible)
   - position: relative (proper stacking)

✅ Containers now have:
   - min-height: 100vh (flexible)
   - overflow-y: auto (scrollable)

✅ Body now has:
   - overflow-x: hidden only
   - Vertical scroll allowed

🔄 USER WORKFLOW:
================================================================================

PHARMACIST OPENS PRESCRIPTIONS PAGE:
1. Sees statistics at top (Today: X, Week: Y, Total: Z)
2. Can search for specific patient
3. Can filter by date (All, Today, Week, Month)
4. Sees table with all prescriptions

VIEWING PRESCRIPTION DETAILS:
1. Click 👁 "View" button on any row
2. Modal opens showing:
   - Patient name, phone, date
   - Full medicines table with dosage, frequency, duration, qty, price
   - Total amount
   - Doctor's notes
3. Can approve from modal or close

APPROVING PRESCRIPTION:
1. Click ✓ "Approve" button (green)
2. Confirmation dialog shows:
   - Patient name
   - Number of medicines
   - Total amount
3. Click "OK" to confirm
4. System:
   - Creates pharmacy record
   - Reduces stock automatically
   - Shows success message
   - Refreshes table
5. Status changes to "Dispensed"
6. Approve/Delete buttons disappear

DELETING PRESCRIPTION:
1. Click 🗑 "Delete" button (red)
2. Confirmation dialog appears
3. Click "OK" to confirm
4. Prescription is deleted
5. Table refreshes

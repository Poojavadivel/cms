HOW IT WORKS NOW:
================================================================================

1. Doctor clicks "Click to Select Medicine" button
2. Modal opens showing ALL available medicines
3. Each medicine shows:
   - Medicine Name (bold, large text)
   - SKU Code • Price (₹)
4. Doctor clicks on any medicine
5. Medicine is selected automatically
6. Modal closes
7. Selected medicine appears in the button
8. Done! ✅

TECHNICAL DETAILS:
================================================================================

Modal Creation:
- Created dynamically with JavaScript
- Appended to document.body
- Removed after selection

Event Handling:
- Click on button → Opens modal
- Click on medicine → Selects & closes
- Click on × → Closes without selection

Styling:
- Z-index: 10002 (above intake form)
- Max-height: 80vh (scrollable)
- Max-width: 600px
- Smooth transitions

Data Flow:
- Medicine ID stored in row.medicineId
- Medicine name stored in row.medicine
- Price stored in row.price
- All data passed to backend on save

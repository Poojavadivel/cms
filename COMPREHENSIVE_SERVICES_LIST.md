## 🚀 How to Seed

### Method 1: API Call (Recommended)

```bash
POST http://localhost:5000/api/services/seed/initial
Headers:
  x-auth-token: YOUR_AUTH_TOKEN
```

### Method 2: Using Postman/Thunder Client

1. Create new POST request
2. URL: `http://localhost:5000/api/services/seed/initial`
3. Headers: Add `x-auth-token` with your token
4. Send request

### Response:

```json
{
  "message": "200+ comprehensive hospital services seeded successfully",
  "count": 200,
  "breakdown": {
    "Consultation": 16,
    "Procedures": 44,
    "Medication": 39,
    "Lab Tests": 50,
    "Room Charges": 19
  }
}
```

## 🔧 Customization

After seeding, you can:

1. **Update Prices**:
```bash
PUT /api/services/:id
{ "price": 1500 }
```

2. **Add New Services**:
```bash
POST /api/services
{
  "name": "Vitamin C Injection",
  "category": "Medication",
  "price": 150,
  "description": "Vitamin C booster"
}
```

3. **Deactivate Services**:
```bash
DELETE /api/services/:id
```
